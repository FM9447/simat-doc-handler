Simat Doc Handler — Web (Testing)

Lightweight landing page for Chrome testing.

Run locally (Chrome)

From the Flutter app directory run:

flutter pub get
flutter run -d chrome

Open developer tools in Chrome to inspect and debug.

Deploy on GitHub Pages

  1. Go to Settings → Pages in the repository on GitHub.
  2. Set Source to main branch and folder to /docs.
  3. Save — your site will be available at https://<username>.github.io/simat-doc-handler/.
